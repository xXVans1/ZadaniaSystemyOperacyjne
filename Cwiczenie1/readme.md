-> Jak uruchomic programy?:

Aby uruchomic konkretne zadanie nalezy uzyc komend:
* make runA
* make runB
* make runC
* make runD
* make runE

-> Aby wyczyscic katalog z zbednych plikow nalezy uzyc komendy:
* make clean

-> Aby zarchiwizowac oraz zkompresowac katalog nalezy wykonac komende:
* make tar
